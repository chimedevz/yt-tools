Dokumentasi API

      
Panduan lengkap cara menggunakan API dengan cURL dan Node.js.


      
1. Endpoint Cari Video

      
Gunakan endpoint ini untuk mencari video YouTube berdasarkan kata kunci.

      
Endpoint: GET /search
Query Parameter:
- query (string, required): Kata kunci pencarian.

Contoh cURL:
curl -X GET 'https://cryptzx.us.kg/search?query=musik'

Contoh Node.js:
const fetch = require('node-fetch');
(async () => {
  const response = await fetch('https://cryptzx.us.kg/search?query=musik');
  const data = await response.json();
  console.log(data);
})();
      


      
2. Endpoint Unduh Video/Audio

      
Gunakan endpoint ini untuk mengunduh video atau audio dari YouTube.

      
Endpoint: GET /download
Query Parameters:
- url (string, required): URL video YouTube.
- type (string, optional): Jenis file (video atau audio). Default: video.
- quality (string, optional): Kualitas video. Default: 720p.

Contoh cURL:
curl -X GET 'https://cryptzx.us.kg/download?url=https://youtube.com/watch?v=ID&type=video&quality=720p'

Contoh Node.js:
const fetch = require('node-fetch');
(async () => {
  const response = await fetch('https://cryptzx.us.kg/download?url=https://youtube.com/watch?v=ID&type=video&quality=720p');
  const data = await response.json();
  console.log(data);
})();
